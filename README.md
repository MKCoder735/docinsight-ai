# DocInsight AI

AI-powered document summarizer and data analyzer.

## Features

- Document summarization
- CSV analysis
- FastAPI backend
- Next.js frontend
- Docker support

## Project Structure

```text
backend/
frontend/
.github/
docker-compose.yml
```

## Getting Started

### Backend

```bash
pip install -r backend/requirements.txt
uvicorn app.main:app --reload
```

### Frontend

```bash
npm install
npm run dev
```
